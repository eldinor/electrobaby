BabylonPress GLB GLTF Viewer User Guide

BabylonPress GLB GLTF Viewer is a desktop app for opening and inspecting .glb and .gltf models. It supports drag-and-drop, recent files, visual view settings, and camera auto-rotate controls.

Open a Model

  1. Choose File -> Open....
  2. Select a valid .glb or .gltf file.
  3. The model will load in the main viewer window.

You can also press Ctrl+O on Windows or Cmd+O on macOS.

Drag and Drop

Drop a .glb or .gltf file directly into the viewer window to open it.

If the file is not a valid model, the app will show an error message and suggest checking the file.

Recent Files

Use these menu items to reopen work quickly:

  • File -> Open Recent
  • File -> Open Recent Folder
  • File -> Reopen Last On Startup

View Settings

Open Settings -> View... to adjust the presentation of the scene.

  • Tone mapping
  • Exposure
  • Contrast
  • Environment intensity
  • Environment rotation
  • Skybox blur
  • Show environment
  • Background color

Camera Settings

Open Settings -> Camera... to control idle orbit behavior.

  • Auto Rotate
  • Auto Rotate Speed
  • Auto Rotate Delay

Helpful Shortcuts

  • F1: Open this user guide
  • F11: Toggle fullscreen on Windows/Linux
  • Ctrl+Cmd+F: Toggle fullscreen on macOS
  • Esc: Close open settings panels
Troubleshooting: If a model does not load, first confirm that the file is a valid .glb or .gltf. For .gltf, make sure it keeps valid references to its required .bin and texture files. If the screen looks too dark, make sure the environment is enabled in Settings -> View....
